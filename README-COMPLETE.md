# מערכת ניהול תוכן בעברית - Sanity + Next.js

## מה זה?
מערכת שלמה לניהול תוכן בעברית הכוללת:
- **סטודיו Sanity** - ממשק ניהול התוכן (RTL)
- **אתר Next.js** - האתר הציבורי שמציג את התוכן

## הפעלה מהירה

### אופציה 1: הפעלה אוטומטית (מומלץ)
```bash
# הפעל את קובץ ה-PowerShell
./start-both.ps1

# או את קובץ ה-Batch
./start-both.bat
```

### אופציה 2: הפעלה ידנית
```bash
# טרמינל 1 - סטודיו
cd studio
npm run dev

# טרמינל 2 - אתר
cd website  
npm run dev
```

## כתובות

- **סטודיו ניהול**: http://localhost:3333
- **אתר ציבורי**: http://localhost:3000

## איך זה עובד?

### 1. ניהול תוכן (Sanity Studio)
- כנס ל-http://localhost:3333
- צור פוסטים, דפים וקטגוריות
- העלה תמונות
- ערוך הגדרות אתר

### 2. צפייה באתר (Next.js)
- כנס ל-http://localhost:3000
- ראה את התוכן שיצרת
- האתר מתעדכן אוטומטית

## טיפים לשימוש

1. **התחל עם קטגוריות** - צור כמה קטגוריות בסיסיות
2. **כתוב פוסט ראשון** - הוסף תוכן לבדיקה
3. **בדוק באתר** - ראה איך זה נראה ב-localhost:3000
4. **עדכן הגדרות** - מלא את פרטי האתר

## פתרון בעיות

### האתר לא נטען?
- וודא ששני השרתים רצים
- בדוק שהפורטים 3000 ו-3333 פנויים

### אין תוכן באתר?
- צור תוכן בסטודיו קודם
- וודא שהפוסטים פורסמו (יש תאריך פרסום)

### שגיאות חיבור?
- בדוק את קובץ .env בתיקיית studio
- וודא שה-PROJECT_ID נכון
