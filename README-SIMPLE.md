# סטודיו Sanity בעברית - גרסה פשוטה

## מה זה?
סטודיו ניהול תוכן בעברית עם ממשק RTL פשוט וקל לשימוש.

## התקנה מהירה

1. **שכפל את הפרויקט:**
   ```bash
   git clone https://github.com/liormedan/sanity_studio_heb_template.git
   cd sanity_studio_heb_template/studio
   ```

2. **התקן תלויות:**
   ```bash
   npm install
   ```

3. **הגדר פרויקט Sanity:**
   ```bash
   npx sanity init --create-project "שם הפרויקט שלך" --dataset production
   ```

4. **עדכן קובץ .env:**
   ```
   SANITY_STUDIO_PROJECT_ID=הID_שקיבלת
   SANITY_STUDIO_DATASET=production
   ```

5. **הפעל את הסטודיו:**
   ```bash
   npm run dev
   ```
