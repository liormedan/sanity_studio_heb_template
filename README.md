# מערכת ניהול תוכן בעברית 🇮🇱

מערכת שלמה לניהול תוכן בעברית עם Sanity CMS ו-Next.js

## מה זה כולל?

- 📝 **סטודיו Sanity** - ממשק ניהול תוכן בעברית (RTL)
- 🌐 **אתר Next.js** - אתר ציבורי מהיר ורספונסיבי
- 🎨 **עיצוב RTL** - תמיכה מלאה בעברית
- ⚡ **מוכן להעלאה** - Vercel ready

## הפעלה מקומית

```bash
# התקנת תלויות
npm run install:all

# הפעלת שני הפרויקטים יחד
npm run dev
```

או בנפרד:
```bash
# סטודיו בלבד
npm run dev:studio

# אתר בלבד  
npm run dev:website
```

## כתובות מקומיות
- **סטודיו**: http://localhost:3333
- **אתר**: http://localhost:3000
