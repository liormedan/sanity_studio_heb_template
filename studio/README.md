# סטודיו Sanity בעברית

סטודיו ניהול תוכן בעברית עם ממשק RTL.

## הפעלה מקומית
```bash
npm install
npm run dev
```

## העלאה ל-Vercel
1. חבר את הפרויקט ל-Vercel
2. הגדר משתני סביבה:
   - `SANITY_STUDIO_PROJECT_ID=7kzkwqzg`
   - `SANITY_STUDIO_DATASET=production`
3. העלה!

## סוגי תוכן
- פוסטים
- דפים  
- קטגוריות
- הגדרות אתר
